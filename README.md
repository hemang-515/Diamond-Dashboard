# Diamond Dashboard  
A complete Exploratory Data Analysis (EDA) and visualization dashboard created using Python to analyze the **Diamond Dataset**.  
This project helps understand how features like **carat, cut, color, clarity, depth, table, and dimensions** impact the price of diamonds.

---

##  Project Overview

This project focuses on:

- Cleaning and preparing the dataset  
- Performing Exploratory Data Analysis (EDA)  
- Creating visualizations using Matplotlib & Seaborn  
- Understanding feature relationships with price  
- Building a dashboard-like structure inside a Jupyter Notebook  

---

##  Dataset Description

The dataset includes the following columns:

| Feature | Description |
|--------|-------------|
| **carat** | Weight of the diamond |
| **cut** | Quality of cut (Fair, Good, Very Good, Premium, Ideal) |
| **color** | Diamond color graded from D (best) to J (worst) |
| **clarity** | Purity grade (IF, VVS1, VVS2, VS1, VS2, SI1, SI2) |
| **depth** | Depth percentage |
| **table** | Width of top of the diamond |
| **price** | Price in USD |
| **x** | Length (mm) |
| **y** | Width (mm) |
| **z** | Depth (mm) |
